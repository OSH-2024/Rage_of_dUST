## <center> Rage_of_dUST
这里是中国科大 OSH 课程 Rage_of_dUST 小组的大作业仓库，我们的目标是使用 Rust 语言改写 HuaWei LiteOS 的内存管理单元(MMU)，利用 Rust 语言极高的安全性和并行性能，使得 LiteOS 中的 MMU 单元的安全性得到提升。
